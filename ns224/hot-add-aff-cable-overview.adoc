= Cablea una estantería NS224 a tu sistema AFF
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Conecta tu shelf NS224 a tu sistema AFF para proporcionar conectividad multivía que garantice la redundancia del almacenamiento y alta disponibilidad para el par HA.



== Visión general del cableado de los sistemas AFF

El cableado difiere según el modelo de plataforma AFF porque cada sistema tiene asignaciones de puertos y configuraciones de módulos de E/S únicas.

Durante el procedimiento de cableado de incorporación en caliente, conectas cada estante NS224 a ambos controladores del par de HA:

* Cada módulo NSM de estantería se conecta a un puerto de almacenamiento dedicado en cada controlador, creando dos rutas por estantería (una a través de NSM A, otra a través de NSM B).
* Esta configuración de doble ruta proporciona redundancia de almacenamiento y garantiza el acceso continuo a los datos si falla una sola ruta.
* Los cables se conectan desde los puertos e0a/e0b (NSM100) o e1a/e1b (NSM100B) del estante NSM a los puertos de almacenamiento compatibles con RoCE de los controladores.




== Recursos adicionales

Tu sistema de hardware puede ser compatible tanto con las estanterías NS224 con módulos NSM100 como con las estanterías NS224 con módulos NSM100B. Para obtener información completa sobre los nombres de los puertos y los detalles de configuración, consulta https://hwu.netapp.com["Hardware Universe de NetApp"^].
