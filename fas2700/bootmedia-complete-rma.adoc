= Devuelva la pieza con el error a NetApp - AFF A220 y FAS2700
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Devuelva la pieza que ha fallado a NetApp, como se describe en las instrucciones de RMA que se suministran con el kit. Consulte https://mysupport.netapp.com/site/info/rma["Devolución de piezas y sustituciones"] la página para obtener más información.

El sistema FAS2700 solo admite procedimientos de recuperación de medios de arranque manuales. No se admite la recuperación automática de medios de arranque.
