= Devuelva la parte que ha fallado a NetApp - AFF A200
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Devuelva la pieza defectuosa a NetApp después de reemplazar el medio de arranque, como se describe en las instrucciones de RMA enviadas con el kit. Ver el https://mysupport.netapp.com/site/info/rma["Devolución de piezas y sustituciones"] página para más información. El sistema AFF A200 solo admite procedimientos de recuperación de medios de arranque manuales. No se admite la recuperación automática de medios de arranque.
