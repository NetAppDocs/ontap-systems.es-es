= Devuelva la pieza con el fallo a NetApp - AFF A700s
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Devuelva la pieza defectuosa a NetApp, como se describe en las instrucciones de RMA enviadas con el kit. Ver el https://mysupport.netapp.com/site/info/rma["Devolución de piezas y sustituciones"] página para más información. El sistema AFF A700s solo admite procedimientos de recuperación de medios de arranque manuales. No se admite la recuperación automática de medios de arranque.
