= Devolver el medio de arranque fallido a NetApp - AFF A800
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Si un componente de su sistema de almacenamiento AFF A800 falla, devuélvalo a NetApp. Ver el https://mysupport.netapp.com/site/info/rma["Devolución de piezas y sustituciones"] página para más información.
