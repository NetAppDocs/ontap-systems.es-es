= Sustituye en frío una bandeja DS212C, DS224C o DS460C
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Debes realizar una sustitución en frío del shelf cuando cambies un shelf de unidades en un sistema de producción que tenga discos en uso. Este es un procedimiento disruptivo que requiere que detengas los controladores en tu par HA.

Este procedimiento se aplica a las estanterías con módulos IOM12, IOM12B o IOM12C.

Use el artículo de la base de conocimientos de NetApp https://kb.netapp.com/onprem/ontap/hardware/How_to_replace_a_shelf_chassis_using_a_cold_shelf_removal_procedure["Cómo sustituir un chasis de bandeja mediante un procedimiento de retirada de bandeja de datos fríos"].
